# Overview

A short project overview and quick start for the React LMS starter.

## What is this project?

A lightweight React + Vite starter for a Library Management System with an admin dashboard and user pages (login, signup, browse books, profile).

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173 to view the app.

## Where to look next

- `src/` — application source
- `public/` — static assets (CSS, JS, images)
- `docs/` — this folder with short reference docs
